# Blog MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a static blog (index + 1 inaugural article on Ontario commercial build-outs) integrated into the existing Platinum site, with full SEO/JSON-LD metadata.

**Architecture:** Hand-authored static HTML — `blog.html` (magazine index, listing articles inline) + `blog/<slug>.html` per article. New CSS section in the existing `assets/css/main.css`. No new JS. No build step. Vercel clean URLs already enabled.

**Tech Stack:** HTML5, vanilla CSS (custom properties + cubic-bezier transitions, Oswald + Inter fonts), Vercel static hosting.

**Spec:** [docs/superpowers/specs/2026-04-29-blog-mvp-design.md](../specs/2026-04-29-blog-mvp-design.md)

**Note on TDD:** This is a static HTML/CSS project with no test runner. "Tests" here are visual verifications in `npx live-server --port=5500` and view-source checks for meta tags. Each task ends with explicit verification steps before the commit.

## Task 1: Add `Blog` nav link sitewide

**Files:**
- Modify: `index.html`, `about.html`, `services.html`, `gallery.html`, `virtual-tours.html`, `environmental-policy.html`, `contact.html`, `quote.html`

**Why:** Every existing page exposes a horizontal nav (`.nav-desktop`) and a mobile overlay (`.nav-overlay > ul`). The new Blog page needs an entry in both, on every page, with `class="active"` set only on the new blog pages (later tasks).

- [ ] **Step 1: Add Blog link to `.nav-desktop` in each existing page**

For every existing HTML file listed above, locate the `<nav class="nav-desktop">` block and insert a `Blog` link **between** `Gallery` and `Virtual Tours`. Example (from `index.html`):

Before:
```html
      <a href="gallery.html">Gallery</a>
      <a href="virtual-tours.html">Virtual Tours</a>
```

After:
```html
      <a href="gallery.html">Gallery</a>
      <a href="blog.html">Blog</a>
      <a href="virtual-tours.html">Virtual Tours</a>
```

If a page has the `class="active"` on its own link (e.g. `virtual-tours.html` has `<a href="virtual-tours.html" class="active">`), do NOT touch that — only add the new Blog line.

- [ ] **Step 2: Add Blog link to `.nav-overlay > ul` in each existing page**

Same logic for the mobile overlay menu. Locate the `<div class="nav-overlay">` → `<ul>` block and insert:

Before:
```html
    <li><a href="gallery.html">Gallery</a></li>
    <li><a href="virtual-tours.html">Virtual Tours</a></li>
```

After:
```html
    <li><a href="gallery.html">Gallery</a></li>
    <li><a href="blog.html">Blog</a></li>
    <li><a href="virtual-tours.html">Virtual Tours</a></li>
```

- [ ] **Step 3: If `quote.html` does not have a sidebar/nav** (verify with `Grep "nav-desktop" quote.html`), skip it. Otherwise apply the same edits.

- [ ] **Step 4: Verify visually**

Run: `npx live-server --port=5500`
Open `http://127.0.0.1:5500/` and click through each page. Confirm `Blog` appears in the desktop nav between Gallery and Virtual Tours. Resize to ≤1024px and open the burger — confirm `Blog` appears in the overlay list too. The link will 404 for now (blog.html doesn't exist yet) — expected.

- [ ] **Step 5: Commit**

```bash
git add index.html about.html services.html gallery.html virtual-tours.html environmental-policy.html contact.html quote.html
git commit -m "Add Blog nav link sitewide (placeholder, target page next)"
```

    <div class="article__body">
      <p>Opening a new commercial space — whether it's a coffee shop, a restaurant, a retail store, or a multi-tenant plaza unit — is one of the biggest investments a tenant or franchisee will ever make. The construction phase alone can run anywhere from eight weeks to seven months depending on scope. For first-time operators, the gap between "lease signed" and "doors open" is often where surprises and budget overruns hide.</p>
      <p>This guide walks through the five phases of a typical Ontario commercial build-out, what to expect at each stage, and the pitfalls that consistently catch new tenants off guard.</p>

      <h2>Step 1 — Lease &amp; landlord coordination</h2>
      <p>The build-out actually begins before the lease is signed. The landlord's <strong>work letter</strong> defines what is delivered as base building (typically structure, demising walls, basic HVAC trunk, washroom rough-in) versus what falls to the tenant. Negotiating this scope is the single biggest cost lever you have — anything in the work letter is paid by the landlord, anything outside it is on you.</p>
      <p>Common items to push for: extra electrical capacity, additional HVAC tonnage, demising-wall finishes, exterior storefront. Common items left to the tenant: interior partitions, all finishes, kitchen and serving equipment, signage, fire-suppression upgrades.</p>

      <h2>Step 2 — Design &amp; permit drawings</h2>
      <p>Once the lease is firm, the tenant retains an architect (and usually a mechanical/electrical/plumbing engineer) to produce two sets of drawings: a <strong>design intent</strong> set used for landlord approval, and a more detailed <strong>permit set</strong> that the municipality will review.</p>
      <p>For a typical 2,000–3,500 sq ft restaurant or retail unit, allow four to eight weeks for design. Franchise concepts move faster because the brand provides a prototype design package, but the local engineer still has to adapt mechanical and electrical to the specific suite.</p>
      <p>Three things slow this stage down more than anything else:</p>
      <ul>
        <li>Late decisions on equipment (especially restaurant kitchens — cut sheets drive the entire MEP layout).</li>
        <li>Landlord redlines arriving piecemeal instead of in a single review pass.</li>
        <li>Site conditions discovered late (asbestos in older buildings, undersized electrical service, missing gas service).</li>
      </ul>

      <h2>Step 3 — Permits &amp; approvals</h2>
      <p>In Ontario, a commercial build-out almost always requires a building permit from the municipality. Depending on the use, you may also need:</p>
      <ul>
        <li><strong>Public Health Unit approval</strong> — mandatory for any food-service operation, reviewed alongside or after the building permit.</li>
        <li><strong>TSSA registration</strong> — Technical Standards and Safety Authority, for any gas-fired equipment (kitchen ranges, fryers, water heaters, rooftop units).</li>
        <li><strong>ESA inspection</strong> — Electrical Safety Authority, separate from the building permit, mandatory for all electrical work.</li>
        <li><strong>Signage permit</strong> — usually a separate application, sometimes through a different department.</li>
        <li><strong>Liquor licence (AGCO)</strong> — for restaurants with bar service, runs in parallel and can be the long pole if started late.</li>
      </ul>
      <p>Permit timelines vary widely by municipality. In the GTA, expect three to six weeks from a clean submission to permit-in-hand. Smaller jurisdictions can be faster; larger or busier ones (Toronto, Mississauga) can stretch longer, especially if the application kicks back for revisions. Plan for at least one round of comments — submissions that pass on the first review are the exception, not the rule.</p>

      <h2>Step 4 — Construction phase</h2>
      <p>With permits in hand, the general contractor mobilizes. A typical commercial build-out runs through five overlapping stages:</p>
      <ol>
        <li><strong>Demolition &amp; site prep</strong> — strip back to base building, expose existing services. Often turns up surprises (capped drains in the wrong spot, undersized electrical feeds).</li>
        <li><strong>Framing &amp; rough-in</strong> — partition walls, plumbing and electrical rough, HVAC ductwork, fire-suppression piping. This is the longest single block and where most of the inspections happen.</li>
        <li><strong>Drywall, ceilings, finishes</strong> — closing in the walls, painting, flooring, millwork installation.</li>
        <li><strong>Final MEP &amp; equipment set</strong> — fixtures, light fittings, HVAC commissioning, kitchen equipment installation, gas connections.</li>
        <li><strong>Punch list &amp; deficiency walk</strong> — the tenant walks the space with the GC and lists every item still to be addressed before opening.</li>
      </ol>
      <p>Owner-supplied items (FF&amp;E — furniture, fixtures, equipment) are a perennial schedule risk. Anything the tenant orders directly must arrive on time and to the right spec, or the GC's sequencing falls apart. Restaurant equipment, custom millwork, and signage are the most common culprits.</p>

      <h2>Step 5 — Inspections &amp; occupancy</h2>
      <p>Before opening, the project must pass final inspections from the building department, ESA, the health unit (if applicable), and TSSA (if applicable). Each is independent — passing one doesn't mean the next is automatic.</p>
      <p>Once cleared, the municipality issues an <strong>occupancy permit</strong> (or its equivalent — terminology varies). Only then can the tenant legally open to the public.</p>
      <p>Plan a deficiency window of one to two weeks between substantial completion and the soft-opening date. Trying to open on the same day construction wraps almost always backfires.</p>

      <h2>Realistic timelines</h2>
      <p>Every project is different, but these are typical ranges for Ontario commercial build-outs from <em>permit-in-hand</em> to <em>occupancy</em>:</p>
      <table>
        <thead>
          <tr><th>Project type</th><th>Typical duration</th></tr>
        </thead>
        <tbody>
          <tr><td>Small retail (under 1,500 sq ft)</td><td>8–12 weeks</td></tr>
          <tr><td>Coffee shop or QSR (1,500–2,500 sq ft)</td><td>10–14 weeks</td></tr>
          <tr><td>Full-service restaurant (2,500–4,500 sq ft)</td><td>14–20 weeks</td></tr>
          <tr><td>Multi-unit plaza or larger fit-out</td><td>20–28 weeks</td></tr>
        </tbody>
      </table>
      <p>Add four to ten weeks of design and permitting on top of these. A common rule of thumb: from lease signing to opening day, allow six to nine months for a restaurant, four to six months for retail.</p>

      <h2>Common pitfalls — and how to avoid them</h2>
      <ul>
        <li><strong>Under-scoping the landlord work letter.</strong> Every item missed in the work letter becomes a tenant cost. Walk the site with the GC and the architect <em>before</em> signing the lease.</li>
        <li><strong>Late equipment ordering.</strong> Long-lead items (rooftop HVAC units, custom kitchen equipment, signage) need to be ordered the moment design is locked, not when the GC asks for them.</li>
        <li><strong>Rushing permit drawings.</strong> Submissions with missing details get kicked back, and a single revision round can add three weeks. Investing one extra week on the front end almost always saves time on the back end.</li>
        <li><strong>Skipping the pre-construction meeting with the landlord.</strong> Loading dock access, after-hours work rules, deliveries, dust control — every landlord has rules, and finding out mid-construction is expensive.</li>
        <li><strong>Treating soft-opening as the deadline.</strong> The hard deadline is occupancy permit, not opening day. Build in a deficiency buffer.</li>
      </ul>

      <p>A well-planned build-out is a series of decisions made early, in the right order, with the right people in the room. The construction phase itself is the most visible part — but most of what determines whether a project finishes on time and on budget happens before a single wall is framed.</p>
    </div>
